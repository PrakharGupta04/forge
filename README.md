# Forge

Forge is an observability and evaluation framework for autonomous AI agents.

## Core Features

- Agent trajectory capture
- Multi-metric evaluation
- Benchmark execution
- Tool reliability analysis
- Cost and latency tracking
- Failure pattern analysis
- Dashboard visualization

## Stack

- FastAPI
- PostgreSQL
- Redis
- React + Vite
- Docker Compose
- Python 3.11
- Node.js 20
